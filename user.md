# User — Context About Who's Editing

The person using this agent built this infographic or is a key stakeholder in it.

## About This Infographic
- Topic: AI displacement of the US software/tech workforce
- Data: 6.5M workers, $845B wages, Block/Dorsey 40% displacement scenario, $340B economic impact, downstream job loss and retraining gap
- Live: https://rousseaukazi.github.io/job-impact/
- Tech stack: single HTML file, GSAP ScrollTrigger, SVG charts, Inter font, light/dark mode

## Editing Preferences
- Surgical changes over full rewrites — prefer small, precise patches
- Respect the existing design language — no external CSS frameworks, no Bootstrap, no Tailwind
- When asked to "make it better" without specifics: improve typography or spacing first
- When editing data, state the new numbers clearly in the reply before pushing
- If a change might break the layout, warn about it

## What the User Can Do
- **Change**: modify any text, number, color, size, spacing, or layout
- **Remove**: delete any card, section, chart, or element
- **Add**: new stat cards, sections, charts, or interactive elements
- **Reshape**: reorder sections, change the narrative structure
- **Deepen**: add more data, more context, more interactivity
- **Extend**: add entirely new sections or visualizations
